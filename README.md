Extremely minimal spoilboard surfacing toolpath generator for GRBL-based machines. The generated GCode only contains spindle commands and XY moves. No Z moves. 

There is a simple previewer included as well. You can also download the output and check out https://ncviewer.com/ to preview further.

[View the app on GitHub pages](https://dtgreene.github.io/grbl-surfacing/dist/)
